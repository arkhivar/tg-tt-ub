# Создай Telegram Userbot для сортировки топиков в группах

## Описание проекта
Мне нужен веб-сервис на Flask, который управляет Telegram юзерботом для сортировки топиков в группах. Юзербот работает от имени моего аккаунта @ppss_printer и может сортировать топики в ЛЮБОЙ группе, где я являюсь участником.

## Ключевые требования

### 1. Архитектура
- Flask веб-сервер с простым UI для управления
- Telethon для работы с Telegram API как юзербот (НЕ обычный бот!)
- Использовать telethon.sync для интеграции с Flask [[0]](#__0)
- НЕ использовать threading для Telethon операций - библиотека сама управляет asyncio [[3]](#__3)

### 2. Аутентификация юзербота
ВАЖНО: Это юзербот, не обычный бот! Процесс логина отличается:

1. При первом запуске:
   - Запросить номер телефона (мой номер сохранён в Replit Secrets)
   - Telethon отправит код подтверждения в Telegram
   - Показать форму для ввода кода в веб-интерфейсе
   - После успешного входа создаётся файл сессии (session.session)
   - Этот файл нужно сохранить и переиспользовать!

2. При последующих запусках:
   - Использовать сохранённый файл сессии
   - Автоматический вход без повторной аутентификации

Код для первого входа:

from telethon.sync import TelegramClient
import os
